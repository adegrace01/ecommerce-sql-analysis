Ecommerce Sales Analysis Using SQL

This project analyzes a full year of ecommerce sales data to uncover key business insights.

DATASET
Transactional ecommerce sales data containing:
1.Order ID
2.Product
3.Quantity Ordered
4.Price
5.Order Date
6.Purchase Address

Project Workflow
*Database creation
*Data import from CSV files
*Data cleaning and transformation
*Revenue calculation
*Business analysis using SQL queries

Key Business Questions Answered
1.What was the best sales month?
2.Which products generated the most revenue?
3.Which cities generated the highest sales?
4.What time of day do customers buy the most?
5.Which products are frequently bought together?

Tools Used
*SQL (MySQL)
*Data Cleaning
*Data Analysis# ecommerce-sql-analysis
